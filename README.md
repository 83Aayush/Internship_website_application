# ARTIUS INTERIOR — WEBSITE PACKAGE

A modern responsive interior design business website built using **Pure HTML, CSS, and JavaScript**.
No backend, server setup, or database is required.

---

# 📁 Project Structure

```bash
ARTIUS-INTERIOR/
│
├── index.html          # Main website file
├── css/
│   └── style.css       # Website styling
│
├── js/
│   └── main.js         # Navigation, animations & contact form logic
```

---

# 🌐 Features

* Fully responsive modern design
* Smooth scrolling navigation
* Animated sections and UI interactions
* Product showcase sections
* "How We Work" process section
* Team testimonials
* Working enquiry/contact form
* Mobile-friendly layout
* Pure frontend project (easy hosting)

## ✅ Setup Steps

### 1. Create Access Key

Visit:

https://web3forms.com

Enter:

```text
77aayushkumar@gmail.com
```

Click **"Create Access Key"**

---

### 2. Check Gmail Inbox

Web3Forms will send an **Access Key** to your email.

Example:

```text
a1b2c3d4-e5f6-7890-abcd-ef1234567890
```

---

### 3. Open `index.html`

Find this line:

```html
<input type="hidden" name="access_key" value="YOUR_WEB3FORMS_ACCESS_KEY">
```

---

### 4. Replace the Placeholder

Example:

```html
<input type="hidden" name="access_key" value="a1b2c3d4-e5f6-7890-abcd-ef1234567890">
```

---

### 5. Save the File

Done ✅

Now every enquiry submitted from the website will be delivered directly to:

```text
77aayushkumar@gmail.com
```

# 🔒 Is the Access Key Safe?

Yes.

Web3Forms access keys are designed to be public and safely used inside HTML files.

The key:

* Cannot access your Gmail
* Cannot send emails from your account
* Can only deliver form submissions TO your email
